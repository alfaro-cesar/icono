# ICONO 👕🔥

Tienda online moderna de ropa desarrollada con React, Vite, Node.js, Express y MySQL.

## 📌 Descripción

ICONO es una tienda virtual enfocada en la venta de ropa para hombres, mujeres y niños.

## 🚀 Tecnologías utilizadas

- html
- javascript
- css


## ✨ Funcionalidades

- Catálogo de productos
- Carrito de compras
- Diseño responsive
